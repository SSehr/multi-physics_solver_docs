bool material::store_property ( std::string  label,

std::string  in_val

Function used to parse from plain text input files and store material properties correctly. This is only used if the framework is incorporated using the API. The label is used to differentiate between different materials. For example, the oxide would have have a label of "SiO2" and the in_var_name is "E_SiO2" The function returns true if in_var_name ends with label and in_val is properly stored. Otherwise, false is returned.
